# Project Title

A fullstack Blog tool

## Run Locally

Clone the project

```bash
  git clone https://github.com/Pratik881/Blog.git
```

Go to the project directory

```bash
  cd Blog 
```
Frontend
```bash
  cd Frontone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
Backend
```bash
  cd Backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
